### T2-P1-003 | ask-question | `ask_question` 结构化提问（PLAN 模式）

| 字段 | 内容 |
|------|------|
| **优先级** | P1 |

| **阻塞点** | — |
| **架构文档** | [`docs/architecture/tools/ask-question.md`](../../../docs/architecture/tools/ask-question.md)（**已定稿**）；运行时挂接：[`plan-runtime.md` §4.2.2 PR-PLB](../../../docs/architecture/plan-runtime.md#422-pr-plbtodos--createplan--askquestion-工具与计划文件)、[`planner.md`](../../../docs/architecture/tools/planner.md) |

**目标**：在 **`mode == Planning`** 时提供内置工具 **`ask_question`**：模型以结构化单选/多选（每题 2–4 选项 + UI 兜底 `__custom__` 槽）向用户索取决策；**阻塞 await** UI 直至作答或取消；问答写入 transcript `plan.ask_question`；**不**在执行态打断 todos 节奏。

**已定稿选型（§4.1 决策表摘要）**

| 维度 | 拍板结论 | 方案 § |
|------|----------|--------|
| **Q1** 题数 | 单次 `questions.length ∈ [1, 4]` | [§4.1 Q1](../../../docs/architecture/tools/ask-question.md#41-落地选型决策表) |
| **Q2** 选项 | 每题 LLM `options ∈ [2, 4]`（**不含** UI `__custom__`） | §4.1 Q2 |
| **Q3** 多选 | `allow_multiple: bool`，默认 false | §4.1 Q3 |
| **Q4** 可见性 | **仅** `Planning` catalog | §4.1 Q4 |
| **Q5** 阻塞 | `requires_user_interaction = true`，await UI | §4.1 Q5 |
| **Q6** transcript | 单条 `plan.ask_question`（题 + 答 + `picked_recommended`） | §4.1 Q6 |
| **Q7** 自定义槽 | UI 自动追加 `id=__custom__`；LLM **禁止**声明 | §4.1 Q7 |
| **Q8** 推荐项 | 每题**恰好一个** `recommended: true`；UI 后缀「— 推荐」 | §4.1 Q8 |
