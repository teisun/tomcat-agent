**目标**：Bash 与长任务（`>10s`）可后台执行，不阻塞主 Agent 推理回合；由 **LLM 多轮** 调 `task_output` / `task_stop` / `task_list` 自驱收尾；规范写入 **bash 架构文档**（本仓库以 [bash.md](../../../docs/architecture/tools/bash.md) 为 SSoT，未另建 `coding-style.md` 专章——若需迁入 `Codeing&Architecture_Spec.md` 可另开 DOC 小卡）。

**子项（与初版设计的映射）**：
- [x] **Bash 后台**：初版写 `detached: true`；现网为 **`run_in_background: true`**（catalog / `tool_exec`），`BashTaskRegistry::spawn` 内 **`tokio::spawn`** 泵 stdout/stderr 写日志，**当前 tool 轮次立即返回** `task_id` + `log_path`（见 `src/core/agent_loop/tool_exec.rs`、`src/core/tools/primitive/bash_task.rs`）。
- [x] **进度 / 取消**：初版 `LongRunningTask` trait **未**引入；等价能力为 **`task_output`（增量日志）**、**`task_stop`（killpg）**、**`task_list`（状态枚举）** 三工具 + 磁盘日志。
- [x] **主循环**：后台路径**不**在 `execute_tool` 内 `await` 子进程 `wait`；子进程生命周期在 registry 守护 task 中；主 agent loop 行为与 **T2-P0-016** 一致。
- [x] **文档**：**[bash.md](../../../docs/architecture/tools/bash.md)** §2.4.4–§2.4.5、§7.2、多轮 LLM 视角 ASCII；**未**实现初版「订阅 `EventBus::TaskProgress`」——当前无该事件面。

**依赖**：T2-P0-001

**被依赖**：T2-P1-006（规范先落）— 本卡 DONE 后该依赖视为已满足其「后台 bash 叙事」最小集；若 T2-P1-006 仍需泛型长任务抽象，可另开任务。
